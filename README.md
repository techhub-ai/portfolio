# Erwin Dela Cruz — Portfolio

Personal portfolio site built with React, TypeScript, and Tailwind CSS v4. Showcases my experience, projects, and skills as a full-stack developer.

## About

I'm a full-stack developer with 8+ years of experience building web and mobile applications — from enterprise banking systems to freelance end-to-end products. Previously at Metrobank and Computer Professionals, Inc., now working independently as a freelance developer.

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS v4, Framer Motion
- **Build tool:** Vite

## Running Locally

**Prerequisites:** Node.js

```bash
npm install
npm run dev       # http://localhost:3000
npm run build     # Production build
npm run preview   # Preview production build
```

## Structure

- `src/App.tsx` — all UI sections (navbar, hero, about, projects, skills, contact)
- `src/constants.tsx` — portfolio data (projects, skills, experience)
- `src/index.css` — Tailwind CSS entry point
